# Premium Beverage Distributor Website

Professional beverage distributor website built using:

- HTML5
- CSS3
- Vanilla JavaScript
- Local JSON inventory system

## Features

- Dynamic product rendering from JSON
- Horizontal category sliders
- Mobile-first responsive design
- WhatsApp order integration
- Search functionality
- SEO optimized
- Production ready

## Folder Structure

index.html  
style.css  
app.js
products.js
/images  

## Deployment

Can be deployed to:

- Netlify
- Vercel
- Shared hosting (cPanel)
